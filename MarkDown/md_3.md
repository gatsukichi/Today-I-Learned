##Links 에 대하여
```
첫번째로 인라인 링크 라는것이 있다.
[text](Url)
```

[Visit GitHub!](www.github.com)

위에 처럼 내가 원하는 글에 링크를 주는것이다.

마찬가지로 굵은글씨도 가능하다.

이게[**정말**](www.naver.com)되네

```
두번째로 참조링크라고합니다.
[text-link][]


[text-link] : www.google.com
```

여기는 [구글][]
여기는 [네이버][]

[구글]:www.google.com
[네이버]:www.naver.com
